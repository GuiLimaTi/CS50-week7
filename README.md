The projects in this repository cover a range of SQL concepts, from basic queries to complex multi-table analysis.

- Pattern matching

## 📂 Projects

### 🎵 Songs
A project focused on querying a dataset of songs and artists.

**Key concepts:**
- SELECT, WHERE, ORDER BY
- LIMIT and filtering
- Aggregation (AVG)
- Pattern matching (LIKE)
- Subqueries and basic JOINs

📁 Location: `songs/`

---

### 🎬 Movies
A more advanced SQL project using an IMDb-style relational database.

**Key concepts:**
- Complex JOIN operations across multiple tables
- Subqueries for layered filtering
- Aggregation and sorting
- Working with relational schemas

📁 Location: `movies/`

### 🕵️ Fiftyville Investigation
A real-world style data investigation using SQL.

**Objective:**
Identify a thief, their accomplice, and their escape destination by analyzing multiple datasets.

**Key concepts:**
- Multi-step data investigation
- Cross-referencing multiple tables
- Logical filtering and elimination
- Analytical reasoning with SQL

📁 Location: `fiftyville/`
