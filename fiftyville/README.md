# 🕵️‍♂️ CS50 Week 7 – Fiftyville Investigation (SQL)

## 📌 Project Overview
This project is part of Harvard's CS50 course and consists of solving a fictional crime using SQL queries.

A theft took place in the city of Fiftyville, and the goal was to identify:
- The thief
- The accomplice
- The escape destination

All conclusions were derived exclusively through querying a relational database.

---

## 🧠 Investigation Approach

The investigation was conducted step by step using SQL, analyzing multiple interconnected datasets:

- Crime scene reports
- Witness interviews
- Bakery security logs
- ATM transactions
- Phone call records
- Flight schedules
- Passenger lists
- Personal records

Each dataset provided clues that were cross-referenced to narrow down suspects and reconstruct the sequence of events.

---

## 🔍 Key Steps

1. Identified the crime details from the crime scene report
2. Analyzed witness interviews for initial leads
3. Checked bakery security logs within the relevant time window
4. Matched license plates to individuals
5. Investigated ATM withdrawals near the crime location
6. Cross-referenced bank accounts with suspects
7. Analyzed phone calls made shortly after the crime
8. Identified the earliest flight out of Fiftyville
9. Checked passenger lists to confirm the escape route

---

## 🧾 Final Findings

| Role        | Result          |
|------------|----------------|
| 🕵️ Thief    | **Bruce**       |
| 🤝 Accomplice | **Robin**       |
| ✈️ Destination | **New York City** |
